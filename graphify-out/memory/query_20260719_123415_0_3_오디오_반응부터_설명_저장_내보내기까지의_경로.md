# Q: 0.3 오디오 반응부터 설명·저장·내보내기까지의 경로

## Answer

App이 analyzeLocalAudioReactions와 시각·채팅 분석을 병렬 실행한다. audioReactionAnalysis.worker의 AudioFeatureAccumulator가 순차 디코딩 sample을 1초 privacy-safe feature로 만들고 selectAudioReactionHighlights가 반응 후보를 고른다. fuseReactionHighlightCandidates가 오디오·채팅 anchor에 시각 문맥만 더한다. buildHighlightNarrative가 사건·반응·추천 이유를 투영하고 durableAnalysisPayload와 highlightExport가 allowlist 저장·CSV/Markdown/JSON 출력으로 전달한다.
